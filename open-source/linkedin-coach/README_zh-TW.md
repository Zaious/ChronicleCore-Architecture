# 職涯導師 (Career Mentor) - Claude Code 技能擴充

> *「把那些雜亂無章的代工履歷全刪了。2026 年沒有人會花大錢請你刻 UI，他們買的是『治理架構』。」*

## 🌟 簡介

**職涯導師** (`linkedin-coach`) 是一個特化的 Claude Code 技能，由企業級 AI 架構師的個人品牌實戰經歷「通識化」擷取而成。它扮演著一位極度冷酷但實用的職涯戰略顧問，專門幫助技術人員將低價值的「任務型定位」，躍升為高價值的「架構與治理型定位」。

如果你的 LinkedIn 頁面看起來就像一長串枯燥的技術框架清單，這個技能將會無情地撕碎它，並將其重構成一封瞄準 B2B 新創老闆與高階獵頭的「隱形顧問邀約信」。

## 🚀 核心火力

*   **無情裁切過濾器 (Ruthless Pruning Engine)**：冷酷地刪除那些會稀釋你專業權威的過時經歷（例如：已經被 AI 取代的苦力活）。
*   **升維打擊轉換器 (Paradigm Shift Generator)**：提升你的職稱維度。你不再只是一個「開發者 (Developer)」，而是「架構師 (Architect)」或是「特約技術長 (Fractional Officer)」。
*   **雙語狩獵排版 (Bilingual Sniper Formatting)**：為全球演算法打造精準的英文 Headline，並為本土老闆撰寫極具說服力的中文 About 區塊。
*   **高維度價值校準 (High-End Value Alignment)**：將你過去看似混亂的經歷（如：論壇管理、不相干科系），統整編織成一個關於「系統思考與複雜治理」的強大敘事。

## 📦 安裝說明

要將此技能載入你的 Claude Code 工作站，請將 `linkedin-coach` 資料夾複製到專案的 `.agent/skills/` 目錄下（或是你管理 Claude Code 技能的指定位置）。

```bash
# 目錄結構範例
your-project/
├── .agent/
│   └── skills/
│       └── linkedin-coach/
│           ├── SKILL.md
│           ├── assets/
│           │   └── persona.md
│           └── references/
│               └── framework.md
```

## 🛠️ 下令範例

載入完成後，你可以對 Claude 下達以下指令，召喚職涯導師：

*   *「嘿 Claude，這是我的 Notion 舊履歷。麻煩載入 `linkedin-coach` 技能幫我做一次無情裁切。我的目標是鎖定年薪三百萬以上的遠端架構師職缺。」*
*   *「用你們的『高維度定位框架』幫我審查現在的 LinkedIn Headline，我最近收到的都是一些浪費時間的初階工作獵頭訊息。」*
*   *「我以前是專利工程師，現在在寫 AI Agent。幫我用升維轉換器，把這兩個經歷寫成一個有說服力的 B2B 顧問故事。」*

## 📜 框架血統

此技能是私有系統 **ChronicleCore A1 專家系統** 的一次「開源通識化 (Open-Source Generalization)」。它剝離了底層的程式碼與複雜多智能體路由邏輯，純粹提取了 AI 治理架構師在 2026 年市場突圍的「戰略心法」，讓一般開發者也能將這套高級商業邏輯作為通識工具使用。

其底層邏輯融合了高階戰略規劃與侵略性的敘事行銷，並在此統一收斂為「職涯導師」的單一人格。
