## 🌟 Overview

The **Career Mentor** (`linkedin-coach`) is a specialized Claude Code skill derived from an enterprise-grade AI Architect's personal branding framework. It is designed to act as a ruthless career strategist, helping technical professionals elevate their positioning from low-tier tasks to high-value governance and architectural roles.

If you are a talented engineer whose LinkedIn profile reads like a generic laundry list of frameworks, this skill will tear it apart and rebuild it as an "invisible consultation invitation" aimed at founders and recruiters who pay for *systems*, not just *code*.

## 🚀 Features

*   **Ruthless Pruning Engine**: Coldly eliminates obsolete or low-value experiences that dilute your professional authority.
*   **Paradigm Shift Generator**: Elevates your job titles. You are no longer just a "Developer"; you become an "Architect" or a "Fractional Officer."
*   **Bilingual Sniper Formatting**: Crafts targeted English headlines for global algorithms and tailored localized "About" sections to close regional deals.
*   **High-End Value Alignment**: Aligns your past chaos (e.g., community management, unrelated degrees) into a cohesive narrative of "Systems Thinking and Complex Governance."

## 📦 Installation

To inject this skill into your Claude Code workspace, copy the `linkedin-coach` folder into your project's `.agent/skills/` directory (or wherever you manage your Claude Code skills).

```bash
# Example structure
your-project/
├── .agent/
│   └── skills/
│       └── linkedin-coach/
│           ├── SKILL.md
│           ├── assets/
│           │   └── persona.md
│           └── references/
│               └── framework.md
```
