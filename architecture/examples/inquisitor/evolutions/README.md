# Evolutions (Capability Packs)

Modular capability extensions that augment the Inquisitor's base identity. Each pack is independently loadable and does not alter the core SKILL.md definition.

| Pack | Focus |
|------|-------|
| `dlc-ai-security/` | LLM security assessment and red-teaming protocols |
| `dlc-confidence-review/` | Multi-agent parallel review with quantitative scoring |
| `dlc-intent-audit/` | Deep intent identification behind code changes |
| `dlc-quality-governance/` | QA standards and bug lifecycle enforcement |
| `dlc-ethics-evaluation/` | Ethics and safety impact assessment rubrics |
| `dlc-governance/` | Internal audit, risk management, policy enforcement |

> Pack contents are maintained in the private system. This listing reflects the Inquisitor's current capability surface.
